このツールの全体像

📌 このマニュアルの目的

個人懇談の日程調整は、保護者ごとの都合・兄弟関係・担任の判断が絡むため、全自動化には馴染みません。このツールは 「集計・見える化だけ自動化し、最終判断は先生が行う」 設計です。GASの経験がなくても、コードをコピー＆ペーストするだけで動作します。

① 保護者がフォームに希望入力
→
② スプレッドシートに自動蓄積
→
③ GASでカレンダー自動生成
→
④ 先生が確定・調整
→
⑤ お手紙印刷・配布
📋

STEP 1　フォーム作成

Googleフォームで保護者が第一〜第三希望を選べるアンケートを作成します。プルダウン形式なので入力ミスが起きません。

📊

STEP 2　スプレッドシート連携

フォームの回答を自動収集。3つのシート（フォーム回答・カレンダー・確定一覧）を準備します。

⚡

STEP 3　GAS自動生成

コードを貼り付けてボタン一発。希望が重なっているコマを赤くハイライトした調整カレンダーが自動で作られます。

📄

STEP 4　確定・お手紙

先生が確定日程を記入。Googleドキュメントのテンプレートからお手紙を印刷して保護者へ配布します。

！

1

Googleフォームを作成する

🎯 このSTEPのゴール

保護者が第一〜第三希望の日時を選べるフォームを完成させ、配布用リンクを取得します。

1

Googleドライブを開き、フォームを新規作成する

drive.google.com にアクセスし、左上の ＋ 新規 → Googleフォーム をクリックします。

学校のGoogle Workspaceアカウントでログインしておくと、フォームの共有が校内に限定できて安全です。
2

フォームのタイトルと説明を入力する

タイトル欄に 個人懇談 希望日程アンケート と入力します。説明欄には以下のような文章を入力してください。

例）「保護者の皆様、個人懇談の希望日程をご記入ください。ご都合の良い日時を第一〜第三希望の順にお選びください。万が一ご希望に添えない場合は担任よりご連絡します。」
3

質問①「保護者名」を追加する

＋ボタンで質問を追加し、種類を 記述式（短文） に設定します。「必須」をオンにしてください。

質問文の例：「保護者のお名前（例：山田 花子）」
4

質問②「お子様の名前」を追加する

同様に 記述式 で追加します。「必須」をオンにしてください。

質問文の例：「お子様のお名前（例：山田 太郎）」
5

質問③〜⑤「第一〜第三希望」をプルダウンで追加する

種類を プルダウン に設定し、選択肢に下の形式で日時を入力します。この文字列はGASコードと完全に一致させる必要があります。

⚠️ 選択肢の書き方は統一してください。例：10/15(火) 14:00〜
全角・半角、スペースの有無など、あとでGASコードにそのままコピーします。
選択肢の例：
10/15(火) 14:00〜
10/15(火) 14:15〜
10/15(火) 14:30〜
10/16(水) 14:00〜
……（全日程分）
6

質問⑥「備考」を任意で追加する（省略可）

兄弟がいる・仕事の都合など保護者が一言添えられる欄を設けると調整がスムーズです。種類は 記述式（長文）、「必須」はオフに設定します。

7

回答を1回に制限する（推奨）

右上の ⚙️ 設定アイコン → 回答 タブ → 「1回の回答に制限する」 をオンにします。

これでGoogleアカウントでログインした人しか回答できなくなります。学校のアカウントがある場合は合わせて「〇〇小学校のユーザーのみに制限」も設定できます。
8

フォームのリンクを保護者に配布する

右上の 送信 ボタン → 🔗 リンクアイコンを選択 → URLを短縮 にチェック → コピーして保護者に配布します（メール・学校連絡アプリなど）。

✅
確認ポイント：自分でテスト送信して、回答がスプレッドシートに届くか確認してから保護者に配布しましょう。テスト回答は後で削除できます。
2

スプレッドシートを準備する

🎯 このSTEPのゴール

フォーム回答が自動収集されるスプレッドシートに、カレンダー調整表と確定一覧の2シートを追加します。

1

フォームとスプレッドシートを連携する

フォーム編集画面の上部 回答 タブをクリック → 右上の緑のスプレッドシートアイコン（「スプレッドシートにリンク」）をクリック → 新しいスプレッドシートを作成 を選び、名前を 「個人懇談日程調整」 などにして 作成 をクリックします。

これ以降、保護者がフォームを送信するたびに自動でシートに追記されます。
2

シート名を確認・変更する

自動作成されたシートのタブ名（デフォルトは「フォームの回答 1」など）を確認します。後でGASコードに入力するので、わかりやすい名前にしておきましょう。

推奨シート名：フォーム回答
タブを右クリック → 「名前を変更」
3

「カレンダー調整表」シートを追加する

シートタブの左下にある ＋ ボタンをクリックし、新しいシートを追加します。タブ名を 「カレンダー調整表」 に変更します。

このシートはGASが自動で書き込みます。中身は空白のままで大丈夫です。
4

「確定済み一覧」シートを追加する

同様に ＋ ボタンでシートを追加し、タブ名を 「確定済み一覧」 に変更します。

以下のヘッダーを1行目に手入力しておきます：
A1：No. ／ B1：保護者名 ／ C1：お子様の名前 ／ D1：確定日時 ／ E1：希望順位 ／ F1：お手紙 ／ G1：備考
5

フォーム回答シートの列番号を確認する

フォーム回答シートを開き、各列が何番目か確認します（A列＝1、B列＝2…）。GASコードに入力するために必要です。

一般的な列構成例：
A列（1）：タイムスタンプ ／ B列（2）：保護者名 ／ C列（3）：お子様の名前
D列（4）：第一希望 ／ E列（5）：第二希望 ／ F列（6）：第三希望 ／ G列（7）：備考
※ フォームで質問を追加した順に列が作られます。

GASコードを設定する

🎯 このSTEPのゴール

コードをコピペして保存するだけで、スプレッドシートに 📋 懇談ツール メニューが追加されます。ボタン一発でカレンダーが自動生成されます。

①

GASエディタを開く

1

メニューから Apps Script を開く

スプレッドシートのメニューバーから 拡張機能 → Apps Script をクリックします。ブラウザの新しいタブでエディタが開きます。

2

既存のコードを全削除する

最初から表示されている function myFunction() {} を Ctrl+A で全選択し、Delete キーで削除します。

3

下のコードを全文コピーして貼り付ける

下の「📋 全文コピー」ボタンを押してコピーし、エディタの空白部分に Ctrl+V で貼り付けます。

②

コードを貼り付ける（★の箇所だけ変更）

⚠️
フォームの選択肢文字列を DATES・TIME_SLOTS にそのままコピーしてください。一字でも違うとカレンダーに反映されません。

// ══════════════════════════════════════════════
//  個人懇談 カレンダー自動生成スクリプト
//  作成：後藤 壮史（大阪大谷大学 教育学部）
//  作成：後藤 壮史（大阪大谷大学 教育学部）
// ══════════════════════════════════════════════

// ──────────────────────────────────────────────
//  ★ ここだけ変更してください ★
// ──────────────────────────────────────────────
const SHEET_FORM  = 'フォーム回答';      // フォーム回答シートのタブ名
const SHEET_CAL   = 'カレンダー調整表';   // カレンダー調整表シートのタブ名
const SHEET_KAKUTEI= '確定済み一覧';      // 確定済み一覧シートのタブ名

const COL_PARENT = 2;  // 保護者名  が何列目か（A=1, B=2…）
const COL_CHILD  = 3;  // お子様名  が何列目か
const COL_WISH1  = 4;  // 第一希望  が何列目か
const COL_WISH2  = 5;  // 第二希望  が何列目か
const COL_WISH3  = 6;  // 第三希望  が何列目か

// 懇談の日程（フォームのプルダウン選択肢と完全一致させること）
const DATES = [
  '10/15(火)',
  '10/16(水)',
  '10/17(木)',
  '10/18(金)',
];

// 懇談の時間帯（フォームのプルダウン選択肢と完全一致させること）
const TIME_SLOTS = [
  '14:00〜14:15',
  '14:15〜14:30',
  '14:30〜14:45',
  '14:45〜15:00',
  '15:00〜15:15',
  '15:15〜15:30',
];

// ──────────────────────────────────────────────
//  以下は変更不要です
// ──────────────────────────────────────────────

/** スプレッドシートを開いたときにメニューを追加 */
function onOpen() {
  SpreadsheetApp.getUi()
    .createMenu('📋 懇談ツール')
    .addItem('① カレンダーを自動生成・更新', 'buildCalendar')
    .addSeparator()
    .addItem('② 重複チェック',          'checkDuplicate')
    .addItem('③ 未確定の保護者を表示',  'showUndecided')
    .addToUi();
}

/** ① カレンダー自動生成 */
function buildCalendar() {
  const ss   = SpreadsheetApp.getActiveSpreadsheet();
  const form = ss.getSheetByName(SHEET_FORM);
  const cal  = ss.getSheetByName(SHEET_CAL);
  const ui   = SpreadsheetApp.getUi();

  if (!form || !cal) {
    ui.alert('⚠️ シートが見つかりません。\nシート名の設定を確認してください。');
    return;
  }

  // フォーム回答を取得（1行目ヘッダー除く）
  const rows = form.getDataRange().getValues().slice(1);

  // スロットマップを初期化 {日付|時間帯: [{name, child, rank}]}
  const slotMap = {};
  DATES.forEach(d => TIME_SLOTS.forEach(t => {
    slotMap[d + '|' + t] = [];
  }));

  // フォーム回答をスロットに振り分け
  rows.forEach(row => {
    const name  = str(row[COL_PARENT - 1]);
    const child = str(row[COL_CHILD  - 1]);
    const w1    = str(row[COL_WISH1  - 1]);
    const w2    = str(row[COL_WISH2  - 1]);
    const w3    = str(row[COL_WISH3  - 1]);
    if (!name) return;
    [[w1,'◎'],[w2,'○'],[w3,'△']].forEach(([slot,rank]) => {
      if (slot && slotMap[slot] !== undefined)
        slotMap[slot].push({name, child, rank});
    });
  });

  // シートを初期化して描画
  cal.clearContents(); cal.clearFormats();
  drawCalendar(cal, slotMap);
  ui.alert('✅ カレンダーを更新しました！\n\n赤いセルは希望が重なっています。\nどちらかの保護者を隣のコマに移してから確定してください。');
}

/** カレンダーを描画する */
function drawCalendar(sh, slotMap) {
  const C = {
    NAVY:'#1A3A6B', NAVY_L:'#EBF0FA', WHITE:'#FFFFFF',
    RED:'#C0392B',  RED_L:'#FDECEA',
    GRAY:'#4A4A4A', GRAY_L:'#F5F5F5',
    YELLOW:'#FFFDE7', ORANGE:'#E07B00'
  };
  const cols = DATES.length + 1;

  // タイトル行
  const t = sh.getRange(1,1,1,cols);
  t.merge();
  t.setValue('📅 個人懇談 日程調整カレンダー　【先生が確定日程を割り当てる表】');
  style(t,{bg:C.NAVY,fc:C.WHITE,bold:true,sz:13,ha:'left'});
  sh.setRowHeight(1,42);

  // 凡例行
  const leg = sh.getRange(2,1,1,cols);
  leg.merge();
  leg.setValue('◎＝第一希望　○＝第二希望　△＝第三希望　　⚠️ 赤背景＝同一コマに2名以上希望あり（要調整）');
  style(leg,{bg:C.YELLOW,fc:C.ORANGE,sz:10,ha:'left'});
  sh.setRowHeight(2,24);

  // 日付ヘッダー
  sh.setRowHeight(3,32);
  style(sh.getRange(3,1),{val:'時間帯',bg:C.NAVY,fc:C.WHITE,bold:true,sz:11});
  DATES.forEach((d,i) =>
    style(sh.getRange(3,i+2),{val:d,bg:C.NAVY,fc:C.WHITE,bold:true,sz:12})
  );

  // データセル
  TIME_SLOTS.forEach((time,ti) => {
    const row = ti + 4;
    sh.setRowHeight(row,52);
    style(sh.getRange(row,1),{val:time,bg:C.NAVY_L,fc:C.NAVY,bold:true,sz:10});
    DATES.forEach((date,di) => {
      const entries = slotMap[date+'|'+time] || [];
      const cell    = sh.getRange(row,di+2);
      const text    = entries.map(e => e.name+'（'+e.rank+'）').join('\n');
      const isDup   = entries.length >= 2;
      const isEmpty = entries.length === 0;
      cell.setValue(text).setWrap(true)
          .setVerticalAlignment('top').setFontSize(10).setHorizontalAlignment('left');
      cell.setBackground(isDup ? C.RED_L : isEmpty ? C.GRAY_L : C.WHITE);
      cell.setFontColor(isDup ? C.RED : C.GRAY);
      if (isDup) cell.setFontWeight('bold');
    });
  });

  sh.setColumnWidth(1,115);
  DATES.forEach((_,i) => sh.setColumnWidth(i+2,185));
}

/** セルスタイル一括適用ヘルパー */
function style(r,{val,bg,fc,bold=false,sz=10,ha='center'}={}) {
  if (val  !== undefined) r.setValue(val);
  if (bg)   r.setBackground(bg);
  if (fc)   r.setFontColor(fc);
  r.setFontWeight(bold ? 'bold' : 'normal').setFontSize(sz)
   .setHorizontalAlignment(ha).setVerticalAlignment('middle')
   .setFontFamily('メイリオ');
}

/** 文字列変換ヘルパー */
function str(v) { return (v || '').toString().trim(); }

/** ② 重複チェック */
function checkDuplicate() {
  const cal  = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_CAL);
  const dups = [];
  TIME_SLOTS.forEach((time,ti) =>
    DATES.forEach((date,di) => {
      const v = cal.getRange(ti+4,di+2).getValue().toString();
      if ((v.match(/（[◎○△]）/g)||[]).length >= 2)
        dups.push(date+' '+time);
    })
  );
  SpreadsheetApp.getUi().alert(
    dups.length
      ? '⚠️ 重複コマ：\n\n'+dups.join('\n')
      : '✅ 重複なし！'
  );
}

/** ③ 未確定の保護者を表示 */
function showUndecided() {
  const ss      = SpreadsheetApp.getActiveSpreadsheet();
  const calText = ss.getSheetByName(SHEET_CAL)
                    .getDataRange().getValues().flat().join(' ');
  const rows    = ss.getSheetByName(SHEET_FORM)
                    .getDataRange().getValues().slice(1);
  const undet   = rows.map(r => str(r[COL_PARENT-1]))
                       .filter(n => n && !calText.includes(n));
  SpreadsheetApp.getUi().alert(
    undet.length
      ? '⬜ 未反映の保護者：\n\n'+undet.join('\n')
      : '✅ 全員カレンダーに反映済み！'
  );
}

日程を確定してお手紙を作成する

🎯 このSTEPのゴール

カレンダー調整表を見ながら全員の日程を確定し、保護者に渡すお手紙を印刷します。

①

日程を確定する

1

カレンダー調整表で重複を解消する

「📋 懇談ツール」→「① カレンダーを自動生成・更新」を実行後、赤くなっているコマを確認します。どちらかの保護者を第二・第三希望のコマに移動します（セルの内容を手で書き換えます）。

兄弟がいる・仕事の都合・前後に別の保護者がいるなど、先生の判断が必要な要素はここで考慮してください。
2

「確定済み一覧」シートに記入する

入力する列：保護者名 ／ お子様の名前 ／ 確定日時 ／ 希望順位（◎○△）
「希望順位」列はどの希望で確定できたかを記録しておくと、保護者への説明に役立ちます。
3

「② 重複チェック」で最終確認する

「📋 懇談ツール」→「② 重複チェック」を実行し、重複がないことを確認します。「✅ 重複なし！」と表示されれば完了です。

保護者数が20名以上の場合は、Googleドキュメントのアドオン「差し込みドキュメント（Mail Merge）」を使うと一括で差し込み印刷できます（無料プランあり）。

次年度の運用：GASコードの DATES・TIME_SLOTS を新しい日程に書き換えるだけで使いまわせます。フォームのプルダウン選択肢も同じ日程に揃えれば完成です。

/** ④ お手紙PDF一括生成メイン */
function exportAllLettersPDF() {
  const ui = SpreadsheetApp.getUi();
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sh = ss.getSheetByName(SHEET_KAKUTEI);

  if (!sh) {
    ui.alert('⚠️ 確定済み一覧シートが見つかりません。');
    return;
  }

  // 確定済み（日時が入っている）行だけ取得
  const rows = sh.getDataRange().getValues().slice(1)
    .filter(r => str(r[PDF_COL_DATE - 1]));

  if (rows.length === 0) {
    ui.alert('⚠️ 確定済みの保護者がいません。\n確定済み一覧の「確定日時」列を確認してください。');
    return;
  }

  // 出力フォルダを取得または作成
  const folder = getOrCreateFolder(OUTPUT_FOLDER_NAME);

  // テンプレートを確認
  let tmplDoc;
  try {
    tmplDoc = DocumentApp.openById(TEMPLATE_DOC_ID);
  } catch(e) {
    ui.alert('⚠️ テンプレートが開けません。\nTEMPLATE_DOC_ID を確認してください。');
    return;
  }

  // 1人ずつPDFを生成してblob配列に収集
  const pdfBlobs = [];
  rows.forEach((row, idx) => {
    const parent = str(row[PDF_COL_PARENT - 1]);
    const child  = str(row[PDF_COL_CHILD  - 1]);
    const date   = str(row[PDF_COL_DATE   - 1]);

    // テンプレートをコピーして差し込み
    const copy   = DriveApp.getFileById(TEMPLATE_DOC_ID).makeCopy('tmp_'+idx, folder);
    const doc    = DocumentApp.openById(copy.getId());
    const body   = doc.getBody();

    body.replaceText('\\{\\{保護者名\\}\\}', parent);
    body.replaceText('\\{\\{児童名\\}\\}',   child);
    body.replaceText('\\{\\{確定日時\\}\\}', date);
    doc.saveAndClose();

    // PDFとして取得してからコピーファイルを削除
    const pdfBlob = copy.getAs('application/pdf').setName(parent + '.pdf');
    pdfBlobs.push(pdfBlob);
    copy.setTrashed(true);  // 一時ファイルを削除
  });

  // 全員分のPDFを結合して保存
  const merged = mergePDFs(pdfBlobs);
  const existing = folder.getFilesByName(OUTPUT_FILE_NAME);
  while (existing.hasNext()) existing.next().setTrashed(true);  // 旧ファイル削除
  const saved = folder.createFile(merged.setName(OUTPUT_FILE_NAME));

  // 確定済み一覧の「お手紙」列を更新
  const allRows = sh.getDataRange().getValues();
  allRows.forEach((r, i) => {
    if (i === 0) return;
    if (str(r[PDF_COL_DATE - 1])) {
      sh.getRange(i + 1, PDF_COL_LETTER).setValue('✅ 出力済');
    }
  });

  ui.alert(
    '✅ PDF生成完了！\n\n'
    + rows.length + '名分のお手紙をまとめました。\n\n'
    + '保存先：Googleドライブ →「' + OUTPUT_FOLDER_NAME + '」フォルダ\n'
    + 'ファイル名：' + OUTPUT_FILE_NAME
  );
}

/** PDFを結合する（バイナリ連結方式） */
function mergePDFs(blobs) {
  if (blobs.length === 1) return blobs[0];

  // GASネイティブのPDF結合：各blobをDrive経由で結合
  const tmpFolder = getOrCreateFolder(OUTPUT_FOLDER_NAME + '/_tmp');
  const fileIds   = blobs.map(b => tmpFolder.createFile(b).getId());

  // Drive APIでPDF結合（v2エンドポイント）
  const boundary = '-------314159265358979323846';
  let   body     = '';

  fileIds.forEach(id => {
    const bytes = DriveApp.getFileById(id).getBlob().getBytes();
    DriveApp.getFileById(id).setTrashed(true);
  });
  tmpFolder.setTrashed(true);

  // GASはネイティブPDF結合APIを持たないため
  // 各PDFを1ページずつGoogleドキュメントにエクスポートして結合する方式を使う
  return mergePDFsViaDoc(blobs);
}

/** Googleドキュメント経由でPDFを結合する */
function mergePDFsViaDoc(blobs) {
  const folder  = getOrCreateFolder(OUTPUT_FOLDER_NAME);
  const mergeDoc = DocumentApp.create('_merge_tmp');
  const mergeBody= mergeDoc.getBody();
  mergeBody.clear();

  blobs.forEach((blob, i) => {
    // 各PDFの内容をドキュメントに埋め込む（テキスト抽出方式）
    const tmpFile = folder.createFile(blob);
    const docId   = Drive.Files.insert(
      {title: 'tmp', mimeType: 'application/vnd.google-apps.document'},
      tmpFile.getBlob(),
      {convert: true}
    ).id;
    const srcDoc  = DocumentApp.openById(docId);
    const srcBody = srcDoc.getBody();
    const elems  = srcBody.getNumChildren();
    for (let j = 0; j < elems; j++) {
      mergeBody.appendParagraph(srcBody.getChild(j).asParagraph().copy());
    }
    if (i < blobs.length - 1) mergeBody.appendPageBreak();
    srcDoc.saveAndClose();
    DriveApp.getFileById(docId).setTrashed(true);
    tmpFile.setTrashed(true);
  });

  mergeDoc.saveAndClose();
  const mergedBlob = DriveApp.getFileById(mergeDoc.getId())
                        .getAs('application/pdf');
  DriveApp.getFileById(mergeDoc.getId()).setTrashed(true);
  return mergedBlob;
}

/** フォルダを取得または作成する */
function getOrCreateFolder(name) {
  const iter = DriveApp.getFoldersByName(name);
  return iter.hasNext() ? iter.next() : DriveApp.createFolder(name);
}

よくある質問（FAQ）

Q1

カレンダーに保護者名が表示されない

フォームのプルダウン選択肢の文字列と、GASコードの DATES・TIME_SLOTS の文字列が一字でも違うと反映されません。フォームの選択肢をそのままコードにコピーしてください。全角・半角の違いにも注意が必要です。

Q2

「📋 懇談ツール」メニューが表示されない

スプレッドシートをいったん閉じて開き直してください。また、GASの承認（権限付与）が完了しているか確認してください。

Q3

「このアプリはGoogleで確認されていません」と表示された

「詳細」→「（安全ではないページ）に移動」→「許可」をクリックしてください。これは自分のGoogleアカウントで作ったスクリプトなので問題ありません。Googleが外部公開アプリを審査する仕組みによるもので、内部スクリプトには表示されることがあります。

Q4

保護者が希望を変更したい

フォームの「回答の編集を許可する」をオンにしておけば、送信後に保護者自身が修正できます。フォーム設定（⚙️）→「回答」タブ→「回答の編集を許可する」をオンにしてください。修正後はカレンダーを再生成してください。

Q5

テスト送信した回答を削除したい

フォームの「回答」タブ → 回答一覧の右上「︙」→「すべての回答を削除」で削除できます。スプレッドシート側は対応する行を手動で削除してください。

Q6

懇談の時間が15分ではなく10分や20分にしたい

GASコードの TIME_SLOTS の内容をご自身の時間帯に書き換えてください。フォームのプルダウン選択肢も同じ内容にすれば対応できます。

Q7

曜日や教室が複数ある（並行開催）場合は？

フォームに「担任の先生名」や「組」の選択肢を追加し、スプレッドシートも組ごとにシートを分けるとスムーズです。GASのコードを複数のシートに対応させることもできます（カスタマイズが必要）。
