このツールの全体像

📌 このシステムの設計思想

個人懇談の日程調整は、保護者の都合・兄弟関係・担任の経験的判断が複雑に絡み合うため、完全自動化には馴染みません。このツールは「集計・見える化・文面生成だけ自動化し、最終判断は先生が行う」設計です。GASの経験がなくても、コードをコピー＆ペーストするだけで動作します。

STEP 1　フォーム作成

教務主任が全校共通フォームを作成・配布。希望日程・在籍クラス・兄弟クラス（複数選択可）を収集します。

STEP 2　スプレッドシート

回答を自動収集。教務主任が定員人数を事前入力。6シート構成で教務主任・担任がそれぞれ参照します。

⚡

STEP 3　GAS設定

コードをコピペして1回設定するだけ。6つのメニューで教務主任用・担任用の機能を使い分けます。

📄

STEP 4〜5　確定・配布

担任が日程を確定し、お手紙を印刷配布。兄弟クラスの担任にはチャットで申し送りを行います。

Googleフォームを作成する

🎯 このSTEPのゴール

保護者が「来られない日時」だけを入力するフォームを完成させ、全校に配布します。いつでも可能な保護者は「特になし」を選ぶだけで送信完了。担任は候補枠が少ない保護者から優先的に日程を確定できます。

📋 追加する質問の一覧（全6問）
質問番号質問内容形式必須
①保護者のお名前記述式（短文）必須
②お子様のお名前記述式（短文）必須
③お子様の在籍クラスプルダウン（1年1組〜6年3組）必須
④来られない日時にチェックしてください（複数選択可）チェックボックス（全24スロット＋「特になし（いつでも可）」）必須
⑤留意点があればご記入ください記述式（長文）任意
⑥兄弟が在籍するクラスをすべて選んでくださいチェックボックス（全クラス＋「兄弟はいない」）必須

⚠️ ③のクラスは未提出者集計に、④の不可日はカレンダーの候補枠数計算に、⑥の兄弟情報は申し送り文面の生成に使われます。GASコードの列番号設定と必ず一致させてください。

1

Googleドライブを開き、フォームを新規作成する

drive.google.com にアクセスし、左上の ＋ 新規 → Googleフォーム をクリックします。

学校のGoogle Workspaceアカウントでログインしておくと、フォームの共有が校内に限定できて安全です。作成者は教務主任が担当します。
2

タイトルと説明を入力する

タイトル欄に 個人懇談 日程アンケート と入力します。説明欄には以下のような文章を入力してください。

例）「個人懇談の日程調整にご協力ください。来られない日時にチェックを入れてください。いつでも都合がつく場合は「特になし（いつでも可）」を選んで送信してください。締切：〇月〇日（〇）」
3

質問①②「保護者名・お子様の名前」を追加する

＋ボタンで質問を追加し、種類を 記述式（短文） に設定します。どちらも「必須」をオンにしてください。

質問①例：「保護者のお名前（例：山田 花子）」
質問②例：「お子様のお名前（例：山田 太郎）」
4

質問③「在籍クラス」をプルダウンで追加する

種類を プルダウン に設定し、選択肢に全クラスを入力します。「必須」をオンにしてください。

選択肢（全18クラス）：
1年1組 / 1年2組 / 1年3組 / 2年1組 / 2年2組 / 2年3組
3年1組 / 3年2組 / 3年3組 / 4年1組 / 4年2組 / 4年3組
5年1組 / 5年2組 / 5年3組 / 6年1組 / 6年2組 / 6年3組
⚠️ GASコードの CLASSES 定数と完全一致させてください。
5

質問④「来られない日時」をチェックボックスで追加する

種類を チェックボックス（複数選択） に設定します。全24スロット＋「特になし（いつでも可）」を選択肢として入力してください。「必須」をオンにします。

選択肢（全25項目）：
□ 10/15(火) 14:00〜14:15　□ 10/15(火) 14:15〜14:30　□ 10/15(火) 14:30〜14:45
□ 10/15(火) 14:45〜15:00　□ 10/15(火) 15:00〜15:15　□ 10/15(火) 15:15〜15:30
□ 10/16(水) 14:00〜14:15　… （同様に10/16・10/17・10/18分も追加）
□ 特になし（いつでも可）
⚠️ 各スロットの文字列はGASコードの DATES・TIMES 定数と完全一致させてください。「特になし（いつでも可）」は NG_NONE 定数と一致させてください。
💡 「特になし（いつでも可）」を選ぶと全スロットが候補になります。いつでも都合がつく保護者はこれだけ選べば送信完了です。
6

質問⑤「留意点」を追加する（省略可）

種類は 記述式（長文）、「必須」はオフに設定します。「兄弟の懇談と近い時間帯を希望」「仕事の都合で15時以降を希望」などを自由に記入してもらう欄です。

💡 カレンダーでは留意点がある保護者に ★ マークが付きます。内容は別途「留意点」列に表示されます。
7

質問⑥「兄弟クラス」をチェックボックスで追加する

種類を チェックボックス（複数選択） に設定します。全クラス＋「兄弟はいない」を選択肢として入力してください。「必須」をオンにします。

選択肢（全19項目）：
□ 1年1組〜6年3組（全18クラス）　□ 兄弟はいない
⚠️ 「兄弟はいない」はGASコードの SIB_NONE 定数と完全一致させてください。最大4人兄弟まで対応（複数チェック可）。
8

回答を1回に制限し、締切を設定する

右上の ⚙️ 設定アイコン → 回答 タブ → 「1回の回答に制限する」 をオンにします。締切日はフォームの説明文に明記し、期限後は「回答を受け付けています」をオフにします。

締切後も遅れる保護者がいます。「② 未提出者一覧を生成」で定期的に確認し、個別対応しましょう。
9

フォームのリンクを全校保護者に配布する

右上の 送信 ボタン → 🔗 リンクアイコン → URLを短縮 にチェック → コピーして保護者に配布します（メール・学校連絡アプリなど）。

配布は教務主任が行います。URLは全クラス共通で構いません。回答に「在籍クラス」を含めているためクラスごとに分ける必要はありません。
✅
確認ポイント：自分でテスト送信して、①「特になし」選択時・②複数の不可日選択時・③兄弟複数選択時の3パターンで回答がスプレッドシートに正しく届くか確認してから配布しましょう。テスト回答は後で削除できます。
2

スプレッドシートを準備する

🎯 このSTEPのゴール

フォーム回答が自動収集されるスプレッドシートに、カレンダー調整表と確定一覧の2シートを追加します。

1

フォームとスプレッドシートを連携する

フォーム編集画面の上部 回答 タブをクリック → 右上の緑のスプレッドシートアイコン（「スプレッドシートにリンク」）をクリック → 新しいスプレッドシートを作成 を選び、名前を 「個人懇談日程調整」 などにして 作成 をクリックします。

これ以降、保護者がフォームを送信するたびに自動でシートに追記されます。
2

シート名を確認・変更する

自動作成されたシートのタブ名（デフォルトは「フォームの回答 1」など）を確認します。後でGASコードに入力するので、わかりやすい名前にしておきましょう。

推奨シート名：フォーム回答
タブを右クリック → 「名前を変更」
3

「カレンダー調整表」シートを追加する

シートタブの左下にある ＋ ボタンをクリックし、新しいシートを追加します。タブ名を 「カレンダー調整表」 に変更します。

このシートはGASが自動で書き込みます。中身は空白のままで大丈夫です。
4

「クラス名簿設定」シートを追加する【教務主任が年度はじめに設定】

＋ ボタンでシートを追加し、タブ名を 「クラス名簿設定」 に変更します。A列にクラス名、B列に定員人数（児童数）を入力します。

入力例：
A1：1年1組　B1：28
A2：1年2組　B2：27
A3：1年3組　B3：29
… （全18クラス分）
💡 これが⑤「未提出者一覧を生成」の基準データになります。年度はじめに教務主任が入力し、クラス編成が変わった年度のみ更新してください。
5

「確定済み一覧」シートを追加する【担任が使用】

同様に ＋ ボタンでシートを追加し、タブ名を 「確定済み一覧」 に変更します。

以下のヘッダーを1行目に手入力しておきます：
A1：No. ／ B1：保護者名 ／ C1：お子様の名前 ／ D1：確定日時 ／ E1：希望順位 ／ F1：お手紙 ／ G1：備考
6

フォーム回答シートの列番号を確認する

フォーム回答シートを開き、各列が何番目か確認します（A列＝1、B列＝2…）。GASコードの列番号設定と一致させる必要があります。

フォーム質問①〜⑥の順に列が作られます（A列はタイムスタンプ）：
A列（1）：タイムスタンプ
B列（2）：①保護者名　← COL_PARENT = 2
C列（3）：②お子様の名前　← COL_CHILD = 3
D列（4）：③在籍クラス　← COL_CLASS = 4
E列（5）：④来られない日時（チェックボックス・カンマ区切り）　← COL_NG = 5
F列（6）：⑤留意点　← COL_NOTE = 6
G列（7）：⑥兄弟クラス（チェックボックス・カンマ区切り）　← COL_SIB = 7
⚠️ チェックボックスで複数選択した場合、スプレッドシートには「10/15(火) 14:00〜14:15, 10/16(水) 15:00〜15:15」のようにカンマ区切りで記録されます。GASのparseChecks関数が自動で分割して処理します。

🎯 このSTEPのゴール

// ═══════════════════════════════════════════════
//  ① 全クラスカレンダーを一括生成
//  【使う人】教務主任
//  【タイミング】フォーム締切後
//  処理：フォーム回答を読み取り、クラスごとに
//        タブを作成。候補枠数で色分け、留意点に★
// ═══════════════════════════════════════════════
function buildAllCalendars() {
  const ss     = SpreadsheetApp.getActiveSpreadsheet();
  const formSh = ss.getSheetByName(SHEET_FORM);
  if (!formSh) {
    SpreadsheetApp.getUi().alert('⚠️ フォーム回答シートが見つかりません。');
    return;
  }

  // フォーム回答を読み込みクラスごとに分類
  const rows     = formSh.getDataRange().getValues().slice(1);
  const byClass  = {};
  CLASSES.forEach(c => { byClass[c] = []; });
  rows.forEach(r => {
    const cls = str(r[COL_CLASS-1]);
    if (byClass[cls]) byClass[cls].push(r);
  });

  // 既存のクラスタブを削除して再生成
  CLASSES.forEach(cls => {
    const old = ss.getSheetByName(cls);
    if (old) ss.deleteSheet(old);
  });

  CLASSES.forEach(cls => {
    const sh = ss.insertSheet(cls);
    sh.setTabColor(COLOR.NAVY);
    drawClassCalendar(sh, cls, byClass[cls] || []);
  });

  SpreadsheetApp.getUi().alert(
    '✅ 全'+CLASSES.length+'クラス分のカレンダーを生成しました！\n\n'
    +'🔴 赤＝残り1枠（最優先）\n'
    +'🟠 橙＝残り2〜3枠\n'
    +'🟡 黄＝残り4〜6枠\n'
    +'白＝残り7枠以上（いつでも可含む）\n'
    +'★＝留意点あり\n\n'
    +'各クラスタブを担任と共有してください。\n'
    +'担任はチェックボックスをオンにして確定してください。'
  );
}

/** クラス1つ分のカレンダーを描画する */
function drawClassCalendar(sh, cls, classRows) {
  sh.clearContents(); sh.clearFormats();

  // 各保護者の「来られる（可能な）スロット」を計算
  const members = classRows.map(r => {
    const ngRaw      = str(r[COL_NG  -1]);
    const ngSlots    = parseChecks(ngRaw, NG_NONE);
    const okSlots    = ALL_SLOTS.filter(s => !ngSlots.includes(s));
    const sibClasses = parseChecks(str(r[COL_SIB -1]), SIB_NONE);
    return {
      parent     : str(r[COL_PARENT-1]),
      child      : str(r[COL_CHILD -1]),
      note       : str(r[COL_NOTE  -1]),
      okSlots,
      okCount    : okSlots.length,
      sibClasses,
    };
  });

  // タイトル行
  const cols = TIMES.length + 2;  // 日付列 + 時間帯列 + 時間帯数
  const tR = sh.getRange(1,1,1,cols);
  tR.merge().setValue('📅 '+cls+' 個人懇談 日程調整カレンダー');
  sty(tR,{bg:COLOR.NAVY,fc:'#FFFFFF',bold:true,sz:13,ha:'left'}); sh.setRowHeight(1,40);

  // 凡例行
  const legR = sh.getRange(2,1,1,cols);
  legR.merge().setValue(
    '🔴残り1枠（最優先）　🟠残り2〜3枠　🟡残り4〜6枠　白＝残り7枠以上　'
    +'🟣紫枠＝兄弟申告あり　★＝留意点あり　'
    +'☑確定済み（取消線）'
  );
  sty(legR,{bg:'#FFFDE7',fc:COLOR.ORANGE_TXT,sz:10,ha:'left'}); sh.setRowHeight(2,22);

  // ヘッダー：日付×時間帯のグリッド
  // 行3：日付（DATES）を結合セルで表示
  // 行4：時間帯（TIMES）を表示
  sh.setRowHeight(3,28); sh.setRowHeight(4,28);
  // 左上2セルは空白
  sty(sh.getRange(3,1,2,2),{bg:COLOR.NAVY,fc:'#FFFFFF'});

  DATES.forEach((d,di) => {
    const startCol = 3 + di * TIMES.length;
    const dateR    = sh.getRange(3, startCol, 1, TIMES.length);
    dateR.merge().setValue(d);
    sty(dateR,{bg:COLOR.NAVY,fc:'#FFFFFF',bold:true,sz:11});
    TIMES.forEach((t,ti) => {
      const c = sh.getRange(4, startCol+ti);
      sty(c,{val:t,bg:COLOR.NAVY_L,fc:COLOR.NAVY,bold:true,sz:9});
      sh.setColumnWidth(startCol+ti, 108);
    });
  });
  sh.setColumnWidth(1,20);   // チェックボックス列
  sh.setColumnWidth(2,130);  // 保護者名列

  // 保護者ごとに1行、列は「CB列・名前列・スロット×24列」
  // 各スロットセル：可能なら空白、不可なら "✕" を表示
  const ROW_START = 5;
  members.forEach((m, mi) => {
    const row    = ROW_START + mi;
    sh.setRowHeight(row, 36);

    // 色決定（候補枠数）
    let rowBg, rowFc;
    if      (m.okCount <= 1) { rowBg=COLOR.RED;    rowFc=COLOR.RED_TXT; }
    else if (m.okCount <= 3) { rowBg=COLOR.ORANGE;  rowFc=COLOR.ORANGE_TXT; }
    else if (m.okCount <= 6) { rowBg=COLOR.YELLOW;  rowFc=COLOR.YELLOW_TXT; }
    else                      { rowBg=COLOR.WHITE;   rowFc=COLOR.GRAY_TXT; }

    // チェックボックス列（A列）
    const cbCell = sh.getRange(row, 1);
    cbCell.insertCheckboxes();
    cbCell.setBackground(rowBg);

    // 保護者名列（B列）：留意点あり→★付加、兄弟あり→紫枠
    const nameLabel = (m.note ? '★ ' : '') + m.parent
      + (m.okCount <= 6 ? '【残'+m.okCount+'枠】' : '');
    const nameCell = sh.getRange(row, 2);
    sty(nameCell,{val:nameLabel,bg:rowBg,fc:rowFc,bold:m.okCount<=3,sz:10,ha:'left'});
    if (m.sibClasses.length > 0) {
      const purpleBorder = SpreadsheetApp.newColor().setRgbColor(COLOR.PURPLE).build();
      nameCell.setBorder(null,true,null,null,false,false,
        purpleBorder, SpreadsheetApp.BorderStyle.SOLID_THICK);
    }

    // スロット列：可能→薄色背景、不可→"✕"グレー
    ALL_SLOTS.forEach((slot, si) => {
      const col   = 3 + si;
      const cell  = sh.getRange(row, col);
      const isOk  = m.okSlots.includes(slot);
      if (isOk) {
        sty(cell,{val:'',bg:rowBg,fc:rowFc,sz:10});
      } else {
        sty(cell,{val:'✕',bg:COLOR.GRAY_L,fc:COLOR.GRAY_B,sz:10});
      }
    });
  });

  // 留意点欄を最後の列に追加
  const noteCol = ALL_SLOTS.length + 3;
  sh.setColumnWidth(noteCol, 220);
  sty(sh.getRange(3, noteCol, 2, 1),{val:'留意点',bg:COLOR.NAVY,fc:'#FFFFFF',bold:true,sz:10});
  members.forEach((m, mi) => {
    const row  = ROW_START + mi;
    const cell = sh.getRange(row, noteCol);
    const sibTxt = m.sibClasses.length
      ? '兄弟：'+m.sibClasses.join('・') : '';
    const noteVal = [m.note, sibTxt].filter(Boolean).join(' ／ ');
    const bg  = m.okCount<=1?COLOR.RED:m.okCount<=3?COLOR.ORANGE
                :m.okCount<=6?COLOR.YELLOW:COLOR.WHITE;
    sty(cell,{val:noteVal,bg:bg,fc:COLOR.GRAY_TXT,sz:9,ha:'left'});
    cell.setWrap(true).setVerticalAlignment('middle');
  });
}

// ═══════════════════════════════════════════════
//  onEdit トリガー
//  チェックボックスをオンにした瞬間：
//    1. 同保護者の全スロットセルに取消線
//    2. 確定済み一覧に自動転記
//  チェックをオフにした場合：
//    1. 取消線を解除
//    2. 確定済み一覧から削除
// ═══════════════════════════════════════════════
function onEdit(e) {
  const sh   = e.source.getActiveSheet();
  const cls  = sh.getName();
  if (!CLASSES.includes(cls)) return;  // クラスタブ以外は無視

  const row = e.range.getRow();
  const col = e.range.getColumn();
  if (col !== 1 || row < 5) return;  // A列・5行目以降のみ対象

  const checked   = e.range.getValue() === true;
  const nameCell  = sh.getRange(row, 2);
  const slotRange = sh.getRange(row, 3, 1, ALL_SLOTS.length);
  const noteCell  = sh.getRange(row, ALL_SLOTS.length+3);

  if (checked) {
    // 取消線を引く
    [nameCell, slotRange, noteCell].forEach(r => {
      r.setFontLine('line-through');
      r.setBackground(COLOR.DONE_BG);
      r.setFontColor(COLOR.DONE_TXT);
    });
    // 確定コマを特定（チェック時に確定スロットをポップアップで選ばせる）
    confirmSlotDialog(e.source, sh, row, cls);
  } else {
    // 取消線を解除・背景色を元に戻す
    [nameCell, slotRange, noteCell].forEach(r => {
      r.setFontLine('none');
    });
    removeFromKakutei(e.source, cls, nameCell.getValue().replace(/[★【】\d枠残]+/g,'').trim());
    restoreRowColor(sh, row);
  }
}

/** 確定スロットをポップアップで選択し、確定済み一覧に転記 */
function confirmSlotDialog(ss, sh, row, cls) {
  // 可能スロット（✕でないセル）を収集
  const slotVals  = sh.getRange(row, 3, 1, ALL_SLOTS.length).getValues()[0];
  const okSlots   = ALL_SLOTS.filter((s,i) => slotVals[i] !== '✕');
  const nameRaw   = str(sh.getRange(row, 2).getValue());
  const parentName= nameRaw.replace(/[★【】\d枠残]+/g,'').trim();

  if (okSlots.length === 0) {
    SpreadsheetApp.getUi().alert('⚠️ 候補スロットがありません。確定できません。');
    sh.getRange(row,1).setValue(false);
    return;
  }

  // 候補一覧をポップアップで表示（番号選択方式）
  const list  = okSlots.map((s,i) => (i+1)+'. '+s).join('\n');
  const resp  = SpreadsheetApp.getUi().prompt(
    '📅 確定スロットを選択',
    parentName+' さんの確定日時を番号で入力してください：\n\n'+list,
    SpreadsheetApp.getUi().ButtonSet.OK_CANCEL
  );
  if (resp.getSelectedButton() !== SpreadsheetApp.getUi().Button.OK) {
    sh.getRange(row,1).setValue(false);
    [sh.getRange(row,2), sh.getRange(row,3,1,ALL_SLOTS.length)]
      .forEach(r => r.setFontLine('none'));
    restoreRowColor(sh, row);
    return;
  }

  const idx  = parseInt(resp.getResponseText().trim()) - 1;
  const slot = okSlots[idx];
  if (!slot) {
    SpreadsheetApp.getUi().alert('⚠️ 無効な番号です。もう一度チェックしてやり直してください。');
    sh.getRange(row,1).setValue(false);
    [sh.getRange(row,2), sh.getRange(row,3,1,ALL_SLOTS.length)]
      .forEach(r => r.setFontLine('none'));
    restoreRowColor(sh, row);
    return;
  }

  // 確定済み一覧に転記
  addToKakutei(ss, cls, parentName, str(sh.getRange(row,2).getValue()), slot);
}

/** 確定済み一覧に1行追加 */
function addToKakutei(ss, cls, parent, nameRaw, slot) {
  let sh = ss.getSheetByName(SHEET_KAKUTEI);
  if (!sh) {
    sh = ss.insertSheet(SHEET_KAKUTEI);
    sh.appendRow(['クラス','保護者名','確定日時','お手紙','備考']);
  }
  // 既に同じ保護者が登録済みなら更新
  const data = sh.getDataRange().getValues();
  for (let i = 1; i < data.length; i++) {
    if (str(data[i][0]) === cls && str(data[i][1]) === parent) {
      sh.getRange(i+1,3).setValue(slot);
      return;
    }
  }
  sh.appendRow([cls, parent, slot, '⬜ 未出力', '']);
}

/** 確定済み一覧から1行削除（チェックオフ時） */
function removeFromKakutei(ss, cls, parent) {
  const sh   = ss.getSheetByName(SHEET_KAKUTEI);
  if (!sh) return;
  const data = sh.getDataRange().getValues();
  for (let i = data.length - 1; i >= 1; i--) {
    if (str(data[i][0]) === cls && str(data[i][1]) === parent)
      sh.deleteRow(i+1);
  }
}

/** 行の背景色を候補枠数に応じて復元 */
function restoreRowColor(sh, row) {
  const nameVal  = str(sh.getRange(row,2).getValue());
  const matchOk  = nameVal.match(/【残(\d+)枠】/);
  const okCount  = matchOk ? parseInt(matchOk[1]) : 24;
  let bg, fc;
  if      (okCount <= 1) { bg=COLOR.RED;    fc=COLOR.RED_TXT; }
  else if (okCount <= 3) { bg=COLOR.ORANGE;  fc=COLOR.ORANGE_TXT; }
  else if (okCount <= 6) { bg=COLOR.YELLOW;  fc=COLOR.YELLOW_TXT; }
  else                   { bg=COLOR.WHITE;   fc=COLOR.GRAY_TXT; }
  const totalCols = ALL_SLOTS.length + 3;
  sh.getRange(row,1,1,totalCols).setBackground(bg).setFontColor(fc).setFontLine('none');
}

/** ④ お手紙PDF一括生成メイン */
function exportAllLettersPDF() {
  const ui = SpreadsheetApp.getUi();
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sh = ss.getSheetByName(SHEET_KAKUTEI);

  if (!sh) {
    ui.alert('⚠️ 確定済み一覧シートが見つかりません。');
    return;
  }

  // 確定済み（日時が入っている）行だけ取得
  const rows = sh.getDataRange().getValues().slice(1)
    .filter(r => str(r[PDF_COL_DATE - 1]));

  if (rows.length === 0) {
    ui.alert('⚠️ 確定済みの保護者がいません。\n確定済み一覧の「確定日時」列を確認してください。');
    return;
  }

  // 出力フォルダを取得または作成
  const folder = getOrCreateFolder(OUTPUT_FOLDER_NAME);

  // テンプレートを確認
  let tmplDoc;
  try {
    tmplDoc = DocumentApp.openById(TEMPLATE_DOC_ID);
  } catch(e) {
    ui.alert('⚠️ テンプレートが開けません。\nTEMPLATE_DOC_ID を確認してください。');
    return;
  }

  // 1人ずつPDFを生成してblob配列に収集
  const pdfBlobs = [];
  rows.forEach((row, idx) => {
    const parent = str(row[PDF_COL_PARENT - 1]);
    const child  = str(row[PDF_COL_CHILD  - 1]);
    const date   = str(row[PDF_COL_DATE   - 1]);

    // テンプレートをコピーして差し込み
    const copy   = DriveApp.getFileById(TEMPLATE_DOC_ID).makeCopy('tmp_'+idx, folder);
    const doc    = DocumentApp.openById(copy.getId());
    const body   = doc.getBody();

    body.replaceText('\\{\\{保護者名\\}\\}', parent);
    body.replaceText('\\{\\{児童名\\}\\}',   child);
    body.replaceText('\\{\\{確定日時\\}\\}', date);
    doc.saveAndClose();

    // PDFとして取得してからコピーファイルを削除
    const pdfBlob = copy.getAs('application/pdf').setName(parent + '.pdf');
    pdfBlobs.push(pdfBlob);
    copy.setTrashed(true);  // 一時ファイルを削除
  });

  // 全員分のPDFを結合して保存
  const merged = mergePDFs(pdfBlobs);
  const existing = folder.getFilesByName(OUTPUT_FILE_NAME);
  while (existing.hasNext()) existing.next().setTrashed(true);  // 旧ファイル削除
  const saved = folder.createFile(merged.setName(OUTPUT_FILE_NAME));

  // 確定済み一覧の「お手紙」列を更新
  const allRows = sh.getDataRange().getValues();
  allRows.forEach((r, i) => {
    if (i === 0) return;
    if (str(r[PDF_COL_DATE - 1])) {
      sh.getRange(i + 1, PDF_COL_LETTER).setValue('✅ 出力済');
    }
  });

  ui.alert(
    '✅ PDF生成完了！\n\n'
    + rows.length + '名分のお手紙をまとめました。\n\n'
    + '保存先：Googleドライブ →「' + OUTPUT_FOLDER_NAME + '」フォルダ\n'
    + 'ファイル名：' + OUTPUT_FILE_NAME
  );
}

/** PDFを結合する（バイナリ連結方式） */
function mergePDFs(blobs) {
  if (blobs.length === 1) return blobs[0];

  // GASネイティブのPDF結合：各blobをDrive経由で結合
  const tmpFolder = getOrCreateFolder(OUTPUT_FOLDER_NAME + '/_tmp');
  const fileIds   = blobs.map(b => tmpFolder.createFile(b).getId());

  // Drive APIでPDF結合（v2エンドポイント）
  const boundary = '-------314159265358979323846';
  let   body     = '';

  fileIds.forEach(id => {
    const bytes = DriveApp.getFileById(id).getBlob().getBytes();
    DriveApp.getFileById(id).setTrashed(true);
  });
  tmpFolder.setTrashed(true);

  // GASはネイティブPDF結合APIを持たないため
  // 各PDFを1ページずつGoogleドキュメントにエクスポートして結合する方式を使う
  return mergePDFsViaDoc(blobs);
}

/** Googleドキュメント経由でPDFを結合する */
function mergePDFsViaDoc(blobs) {
  const folder  = getOrCreateFolder(OUTPUT_FOLDER_NAME);
  const mergeDoc = DocumentApp.create('_merge_tmp');
  const mergeBody= mergeDoc.getBody();
  mergeBody.clear();

  blobs.forEach((blob, i) => {
    // 各PDFの内容をドキュメントに埋め込む（テキスト抽出方式）
    const tmpFile = folder.createFile(blob);
    const docId   = Drive.Files.insert(
      {title: 'tmp', mimeType: 'application/vnd.google-apps.document'},
      tmpFile.getBlob(),
      {convert: true}
    ).id;
    const srcDoc  = DocumentApp.openById(docId);
    const srcBody = srcDoc.getBody();
    const elems  = srcBody.getNumChildren();
    for (let j = 0; j < elems; j++) {
      mergeBody.appendParagraph(srcBody.getChild(j).asParagraph().copy());
    }
    if (i < blobs.length - 1) mergeBody.appendPageBreak();
    srcDoc.saveAndClose();
    DriveApp.getFileById(docId).setTrashed(true);
    tmpFile.setTrashed(true);
  });

  mergeDoc.saveAndClose();
  const mergedBlob = DriveApp.getFileById(mergeDoc.getId())
                        .getAs('application/pdf');
  DriveApp.getFileById(mergeDoc.getId()).setTrashed(true);
  return mergedBlob;
}

/** フォルダを取得または作成する */
function getOrCreateFolder(name) {
  const iter = DriveApp.getFoldersByName(name);
  return iter.hasNext() ? iter.next() : DriveApp.createFolder(name);
}

よくある質問（FAQ）

Q1

カレンダーに保護者名が表示されない

フォームのプルダウン選択肢の文字列と、GASコードの DATES・TIME_SLOTS の文字列が一字でも違うと反映されません。フォームの選択肢をそのままコードにコピーしてください。全角・半角の違いにも注意が必要です。

Q2

「📋 懇談ツール」メニューが表示されない

スプレッドシートをいったん閉じて開き直してください。また、GASの承認（権限付与）が完了しているか確認してください。

Q3

「このアプリはGoogleで確認されていません」と表示された

「詳細」→「（安全ではないページ）に移動」→「許可」をクリックしてください。これは自分のGoogleアカウントで作ったスクリプトなので問題ありません。Googleが外部公開アプリを審査する仕組みによるもので、内部スクリプトには表示されることがあります。

Q4

保護者が希望を変更したい

フォームの「回答の編集を許可する」をオンにしておけば、送信後に保護者自身が修正できます。フォーム設定（⚙️）→「回答」タブ→「回答の編集を許可する」をオンにしてください。修正後はカレンダーを再生成してください。

Q5

テスト送信した回答を削除したい

フォームの「回答」タブ → 回答一覧の右上「︙」→「すべての回答を削除」で削除できます。スプレッドシート側は対応する行を手動で削除してください。

Q6

懇談の時間が15分ではなく10分や20分にしたい

GASコードの TIME_SLOTS の内容をご自身の時間帯に書き換えてください。フォームのプルダウン選択肢も同じ内容にすれば対応できます。

Q7

曜日や教室が複数ある（並行開催）場合は？

フォームに「担任の先生名」や「組」の選択肢を追加し、スプレッドシートも組ごとにシートを分けるとスムーズです。GASのコードを複数のシートに対応させることもできます（カスタマイズが必要）。
